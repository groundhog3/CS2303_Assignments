◆ destroy()

void destroy ( Stack *  stack)

Function to destroy a stack and return memory resources to heap. Note: the memory for the stack struct used to track configuration of the stack is returned to heap, the memory (void pointer to) elements which might currently exist in the stack are not released back to the stack.

Note: It is the responsibility of the user of the stack to manage elements. Note2: It is the responsibility of the user of the stack to manage the pointer to the stack struct. (The stack struct is returned to the memory heap, but the calling code still has a pointer to the, now freed, stack struct. That pointer should be set to NULL upon return from this function.)

Parameters
stackis a pointer to the stack struct. Invalid upon function return.
Returns
none.
