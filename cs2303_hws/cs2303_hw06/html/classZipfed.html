Method to parse a line of data as read and intialize object instance

This method assumes the input file is a CSV format file of ZIP CODE data as supplied by the federal government at:

http://federalgovernmentzipcodes.us/free-zipcode-database-Primary.csv

The function uses strtok library function to parse the file using commas as tokens.
